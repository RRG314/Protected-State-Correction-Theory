# Security Policy

This repository is primarily a research and documentation repository, but it also contains executable code and a static public workbench.

## What To Report

Please report issues such as:

- workbench vulnerabilities
- malicious or unsafe dependency concerns
- export or share-link behavior that could expose unintended data
- any reproducible issue that could affect users of the published static app

## How To Report

Please do not post exploit details in a public issue first.

Instead, email the maintainer at `sreid1118@gmail.com` with:

- a short description of the issue
- steps to reproduce it
- affected files or modules
- any suggested mitigation if you have one

## Scope Note

This policy applies to repository code, tooling, and the published workbench. It does not imply support for broad infrastructure or hosted service guarantees.
