# Roadmap

## Immediate
1. Strengthen the continuous-time theorem beyond the simple projector generator.
2. Add a worked finite-dimensional disturbance-decoupling control example.
3. Expand the QEC section from the 3-qubit bit-flip example toward a cleaner 5-qubit code treatment.

## Near-Term
1. Formalize category-specific correction-capacity notions rather than chasing a single scalar capacity.
2. Investigate continuous quantum correction / feedback correction as a true bridge topic.
3. Add boundary-condition-sensitive variants of the Helmholtz projection discussion.

## Longer-Term
1. Determine whether OCP can produce a genuinely new no-go theorem beyond the direct-sum/indistinguishability statements.
2. Build a paper-ready theorem about correction generators `K` with `ker(K)=S` and contractive action on `D`.
3. Decide whether the strongest publication is best framed as:
   - a protected-state correction framework,
   - an exact/asymptotic bridge framework,
   - or an impossibility-first design theory for correction architectures.
