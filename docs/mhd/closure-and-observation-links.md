# MHD Closure and Observation Links

The MHD repo and the OCP repo meet at a limited but useful structural point.

## Shared structural question

Which record or reduced architecture preserves the quantity we actually care about?

In MHD closure language that can become:
- which field variables remain exactly representable under the chosen closure ansatz?
- which coefficient changes destroy exactness?
- which residual terms witness the obstruction?

In OCP language that becomes:
- what is protected?
- what record is retained?
- what geometry of the admissible family determines recoverability?

## Honest boundary

The OCP repo does not claim a general recoverability theorem for realistic MHD states.
The useful bridge is only on restricted families where the admissible class and record map are explicit.
