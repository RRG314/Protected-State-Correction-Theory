# Final Geometry Novelty and Usefulness Memo

Date: 2026-04-16

## Executive Verdict

Geometry materially strengthens this repo in specific branches, but not universally.
The strongest outcome of this pass is selective:
- strong positive geometry in projector, bounded-domain, generator, and constrained-observation lanes,
- strong negative geometry against amount-only/rank-only/budget-only claims,
- explicit rejection of universal geometry slogans.

## Geometry Tested

Tested geometry classes in this pass:
- subspace/projector geometry,
- operator/kernel/row-space geometry,
- fiber/quotient geometry,
- domain/boundary geometry,
- gauge/orbit geometry.

## Direct Answers To Required End-State Questions

### 1. Where does geometry actually strengthen the repo?

Strongest gains:
- bounded-domain obstruction/classification (boundary geometry + Hodge compatibility),
- constrained-observation/fiber branch (exactness, anti-classifier theorems, noisy hierarchy),
- asymptotic generator lane (block/spectral geometry and mixing no-go),
- sector overlap no-go and exact sector recovery on supported class.

### 2. Which branches gain real theorem power from geometry?

Highest theorem-power gain:
1. constrained-observation / fiber / unified impossibility (`OCP-030` to `OCP-053` core package),
2. bounded-domain CFD/Hodge lane (`OCP-023`, `OCP-028`, `OCP-044`),
3. asymptotic generator lane (`OCP-013`, `OCP-014`, `OCP-015`, `OCP-020`).

Moderate gain:
- exact finite-dimensional and sector/QEC branches (mostly clean formalization and no-go sharpening).

Limited gain:
- gauge branch (mostly corollary-level fit).

### 3. Does geometry produce any new invariant that matters more than rank/count/budget?

Yes, on supported restricted-linear classes:
- geometric inclusion defect (equivalently row-space residual / principal-angle defect),
- collision-gap geometry as quantitative no-go severity,
- family-enlargement and model-mismatch geometric fragility measures.

These outperform rank/count/budget-only summaries on exactness prediction.

### 4. Which open known problems are now more attackable?

Most attackable now:
1. bounded-domain exactness boundary theorem (domain-compatible Hodge class vs boundary mismatch obstruction),
2. weighted-cost anti-classifier theorem beyond unit-cost fixed-library case,
3. robust noisy/enlarged-family extensions of collision-gap and augmentation logic,
4. perturbation-robust vs perturbation-fragile exactness boundary in restricted-linear geometry.

### 5. Which geometry directions should be rejected?

Reject now:
- universal amount/rank/budget geometry classifiers,
- boundary-oblivious projector geometry,
- universal gauge/orbit unification claims,
- unsupported nonlinear fiber-curvature universals,
- any “everything is quotient geometry” narrative.

### 6. Is there any geometry contribution strong enough for a new paper lane?

Yes, one credible lane:
- a restricted theorem/falsification paper on recoverability limits and anti-classifier geometry,
- centered on `OCP-049`/`OCP-050`/`OCP-051`/`OCP-052`/`OCP-053`,
- with bounded-domain geometry as a second, narrower lane if broader theorem extension succeeds.

## What Survived vs Failed (Condensed)

Survived:
- exact projector geometry,
- boundary-sensitive obstruction geometry,
- row-space/fiber/collision geometric invariants,
- generator block/spectral geometry.

Failed:
- broad universals not supported by branch assumptions,
- geometry as renaming without theorem/no-go gain.

## Novelty Tiering

Repo-new:
- integrated anti-classifier/fragility geometry package with executable witnesses.

Likely literature-known:
- base projector and fiber-factorization structures.

Literature-unclear:
- exact strength of fixed-library budget anti-classifier + family-enlargement theorem packaging.

Plausibly literature-distinct:
- theorem-first cross-branch limits program that preserves branch boundaries while proving anti-universal statements.

## Recommendation

Continue geometry work only in branches where it is already theorem-bearing or obstruction-bearing:
- bounded-domain geometry,
- constrained-observation/fiber geometry,
- asymptotic operator geometry.

Freeze or demote universalizing geometry proposals until they produce explicit branch-valid theorems.
