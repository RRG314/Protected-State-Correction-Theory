# Black-Hole and Cosmology Literature Positioning

Status: exploration, non-promoted.

The literature-risk assessment is straightforward. Most of the BH/cosmology content sits close to established lines: linearized GR gauge structure, gravitational-wave parameter degeneracy, cosmological probe complementarity, BH thermodynamic identities, and standard entropy diagnostics.

The additive value in this repository is mostly packaging and diagnostics. Recoverability language helps organize target-versus-record distinctions, and the generated benchmark tables make degeneracy structure explicit for selected templates. That is useful, but it should be described as branch-specific interpretation and benchmarking, not as new foundational physics.

Safe wording is therefore conservative: known structures are recast with recoverability notation; numerical diagnostics are provided for declared setups; open questions are listed without promotion. Unsafe wording includes any claim of new BH thermodynamic theorems, universal Fisher conservation laws, or resolved cosmological tension theorems.

Placement remains as a narrow physics extension and open-problem lane, separate from the core theorem spine.
