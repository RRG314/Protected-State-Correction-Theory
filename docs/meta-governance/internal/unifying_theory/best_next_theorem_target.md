# Best Next Theorem Target

Date: 2026-04-17

## Selected Top Target

**Continuity-aware exactness/stability theorem over the factorization core**.

## Target Statement (program-level form)

Given a declared admissible family `A`, target `T`, record map `M`, and exact factorization condition (`T = R ∘ M` on `A`), derive explicit quantitative conditions under which perturbations of records/family/model imply controlled target error rather than catastrophic failure, with a sharp boundary tied to collision-gap growth.

## Why this is the best next move

1. It directly upgrades the universal core from binary exact/impossible to a robust quantitative layer.
2. It unifies two strongest branch-limited fragility lanes (family enlargement and model mismatch) under one mathematical objective.
3. It addresses the largest current reviewer vulnerability: "exactness claims without stability guarantees".
4. It has immediate impact on both theorem program and engineering tools (diagnostics and repair recommendations).
5. It is more cross-branch leverage than weighted-cost anti-classifier alone or one more bounded-domain subclass theorem.

## Why other candidates are second-tier right now

- **Weighted-cost anti-classifier theorem:** important, but narrower impact than stability-on-core.
- **Stronger family-enlargement taxonomy:** valuable but partially downstream of stability control.
- **Broader bounded-domain exactness classification:** critical branch work, but less general leverage across ecosystem.
- **Stronger symmetry-quotient theorem:** important for soliton lane, less ecosystem-wide than continuity-aware core refinement.

- if compatibility margin `mu > 0` and perturbation size `epsilon < epsilon*(mu)`, then
- target error is bounded by explicit function `B(mu, epsilon)`;
- if `epsilon` exceeds threshold, explicit collision witness implies nonzero ambiguity floor.
