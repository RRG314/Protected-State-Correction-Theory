“Interaction” is useful only when formalized as:
- explicit roles (`T`, `M`, `C`, domain/symmetry/evolution objects),
- explicit compatibility condition,
- explicit admissible family,
- explicit falsification criteria.

Without these, the term is rhetoric, not theorem content.

3. Quantitative invariant that is both nontrivial and reusable across branches.
4. Strong counterexample pressure showing robustness to domain/symmetry/model changes.
