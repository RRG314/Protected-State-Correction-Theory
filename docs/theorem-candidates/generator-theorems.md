# Generator Theorems

## Plain-Language Summary

This file contains the strongest grounded extension beyond the simple projector flow.

The key question is no longer just whether one specific generator `k P_D` works. It is:

> what conditions on a linear correction generator `K` guarantee that the protected component is left alone while the disturbance component is driven to zero?

That question leads to a real theorem, a strong corollary, and a useful failure criterion.

## Theorem OCP-T3: Invariant-Split Generator Theorem

Let `H = S ⊕ D` be an orthogonal decomposition of a finite-dimensional inner-product space. Let `K : H -> H` be linear and assume:

1. `K s = 0` for every `s in S`.
2. `K(D) ⊆ D`.
3. The restricted flow on `D` generated by `-K_D := -K|_D` is exponentially stable.

Then the flow

```text
x_dot = -Kx
```

is an asymptotic OCP correction flow on `(S,D)`, and for every `x_0 = s + d` one has

```text
x(t) = s + e^{-tK_D} d.
```

In particular:
- the protected component is preserved exactly,
- the disturbance component decays exponentially,
- and `x(t) -> s` as `t -> ∞`.

### Proof Sketch

Assumption (1) implies that `S` lies in the kernel of `K`.
Assumption (2) implies that `D` is invariant under `K`.

So in block form relative to `H = S ⊕ D`, the generator has the structure

```text
K = [[0, 0],
     [0, K_D]].
```

The ODE splits into

```text
s_dot = 0,
 d_dot = -K_D d.
```

Hence `s(t)=s(0)` and `d(t)=e^{-tK_D} d(0)`. Exponential stability of `-K_D` gives the claimed decay.

## Corollary OCP-C2: Self-Adjoint PSD Generator Corollary

Let `K` be self-adjoint and positive semidefinite. Let `S = ker(K)` and `D = S^perp`. Suppose the smallest strictly positive eigenvalue of `K` on `D` is `lambda_* > 0`.

Then the flow `x_dot = -Kx` is an asymptotic OCP correction flow and satisfies

```text
||e^{-tK}x - P_S x|| <= e^{-lambda_* t} ||P_D x||.
```

### Why This Corollary Matters

This is the cleanest grounded continuous-time strengthening in the repository because:
- it uses standard spectral theory,
- it is stronger than the `k P_D` special case,
- and it fits real dissipative correction architectures more naturally.
