# Bounded-Domain Hodge Theorems

## Plain-Language Summary

The bounded-domain story is no longer just a warning.

The repository still rejects the naive transplant of the periodic FFT projector to bounded domains. But that does not mean exact bounded-domain correction is impossible. What survives is a narrower positive result:

- if the protected class is built from boundary-compatible divergence-free modes,
- if the disturbance class is built from boundary-compatible gradient modes,
- and if the correction operator is the orthogonal projector for that same decomposition,

then exact correction survives on that bounded family.

This is standard Hodge logic in substance, but it is a real partial solution to the repository's main bounded-domain open problem.

## Theorem OCP-T7: Boundary-Compatible Finite-Mode Hodge Projection

Let `Ω` be a bounded domain with sufficiently regular boundary, and consider finite families

```text
S_bc = span{ J ∇ψ_1, ..., J ∇ψ_m },
G_0  = span{ ∇φ_1, ..., ∇φ_k },
```

where:
- `J(a,b) = (-b,a)`,
- each `ψ_i` is smooth with vanishing boundary trace,
- each `φ_j` is smooth with vanishing boundary trace.

Assume the ambient inner product is the `L^2(Ω)` velocity inner product.

Then:

```text
S_bc ⟂ G_0.
```

Consequently, for every state

```text
u = s + g,  s in S_bc,  g in G_0,
```

the orthogonal projector `P_bc` onto `S_bc` satisfies

```text
P_bc u = s.
```

So `P_bc` is an exact protected-state correction operator on the bounded, boundary-compatible family `S_bc ⊕ G_0`.

### Proof

Take `u = J ∇ψ` with `ψ|_{∂Ω} = 0` and `g = ∇φ` with `φ|_{∂Ω} = 0`.
Then

```text
∫_Ω u · g dx = ∫_Ω u · ∇φ dx.
```

Integrating by parts gives

```text
∫_Ω u · ∇φ dx = -∫_Ω φ div(u) dx + ∫_{∂Ω} φ (u·n) ds.
```

But `u = J ∇ψ` is divergence-free, so `div(u)=0`. Also `φ|_{∂Ω}=0`, so the boundary term vanishes. Hence

```text
∫_Ω u · ∇φ dx = 0.
```

By linearity, every vector in `S_bc` is orthogonal to every vector in `G_0`. Therefore the orthogonal projector onto `S_bc` annihilates `G_0` and preserves `S_bc`, which gives the exact recovery formula.

## Corollary OCP-C6: Sampled Exactness On The Implemented Discrete Bounded Family

Let the repository's bounded discrete family be formed by sampling finite collections of:
- stream-function modes `sin(mπx) sin(nπy)`,
- Dirichlet potential modes `sin(mπx) sin(nπy)`.

Let the protected sampled velocities be generated by `J ∇ψ` and the disturbance sampled velocities by `∇φ`, and let `P_sampled` be the orthogonal projector onto the sampled protected span.

Then the implemented operator is an exact bounded-domain correction operator on that sampled finite family, up to discretization and numerical linear-algebra error.

### Repository Evidence

The executable report in `src/ocp/cfd.py` and `tests/math/test_cfd_projection.py` verifies:
- divergence of the protected and recovered fields stays at machine scale,
- boundary-normal trace of the protected and recovered fields stays at machine scale,
- orthogonality residual between the sampled protected and disturbance bases stays at machine scale,
- two independent projector constructions agree to machine scale.

## Scope And Limit

What this proves:
- bounded-domain exact correction is real on an explicit boundary-compatible finite-mode Hodge family.

What it does not prove:
- that every bounded-domain pressure projection scheme is exact,
- that every discretization realizes the correct bounded Hodge projector,
- that the full bounded-domain continuous problem is finished.

So this is a real partial solution, not a universal bounded-domain theorem.
