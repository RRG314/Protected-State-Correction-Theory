# Compatibility Invariant Search

Date: 2026-04-17
Pass: Meta-theory extraction

## Objective

Search for explicit compatibility invariants that are stronger than rank/count/budget summaries and not merely renamings of existing row-space statements.

## Candidate invariants tested

## I1. Descriptor-Fiber Mixedness Index (DFMI)

### Definition

Let `W` be a finite witness class with exactness verdict `v: W -> {0,1}` and descriptor map `a: W -> A` (rank tuple, budget tuple, etc.).
For each descriptor value `alpha` define:

- `E_alpha = |{w in W: a(w)=alpha, v(w)=1}|`
- `F_alpha = |{w in W: a(w)=alpha, v(w)=0}|`

Then:

`DFMI(a;W) = |{alpha: E_alpha > 0 and F_alpha > 0}| / |im(a)|`.

### Meaning

- `DFMI = 0`: descriptor fibers are verdict-pure (exact classifier may exist).
- `DFMI > 0`: descriptor fibers are mixed, so perfect descriptor-only classification is impossible.

### Classification

`PROVED USEFUL` (finite witness classes).

---

## I2. Irreducible Descriptor Error Lower Bound (IDELB)

### Definition

`IDELB(a;W) = (sum_{alpha in im(a)} min(E_alpha, F_alpha)) / |W|`.

### Theorem (finite witness lower bound)

For any deterministic classifier `h: A -> {0,1}` using only descriptor `a`,

`Err_W(h) >= IDELB(a;W)`.

Equality is attained by majority vote on each descriptor fiber.

### Proof sketch

Inside each descriptor fiber `a^{-1}(alpha)`, any single label must misclassify at least the minority count `min(E_alpha,F_alpha)`. Summing over fibers gives the lower bound. Majority vote attains this count fiberwise.

### Classification

`PROVED USEFUL` (finite witness classes).

---

## I3. Compatibility Lift (CL)

### Definition

For descriptor `a` and refinement `b` (e.g., add compatibility proxy),

`CL(b | a; W) = IDELB(a;W) - IDELB((a,b);W)`.

### Meaning

How much irreducible classification error drops when compatibility information is added.

### Classification

`PROVED USEFUL` on supported witness sets; branch-transfer remains `CONDITIONAL`.

## Decision

The strongest genuinely new invariant package is:
1. `DFMI` (descriptor-fiber mixedness),
2. `IDELB` (irreducible descriptor-only error lower bound),
3. `CL` (compatibility lift).

These are mathematically explicit, computable, and not reducible to pure rank/count language.
