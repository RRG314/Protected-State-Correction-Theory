## Role Labels

Use these labels when linking from README or overview docs:
- **Canonical**: default entrypoint for current readers.
- **Supporting**: deeper technical expansion of canonical claims.
- **Historical/Internal**: provenance and prior-pass context; not default onboarding.

## Navigation Rule

When two documents overlap, canonical docs should explicitly link to the supporting/historical version instead of competing at equal priority.
