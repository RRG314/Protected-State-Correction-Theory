# Structural Discovery Studio

The Structural Discovery Studio is the primary diagnosis and redesign surface.

Given a declared family, target, and record, it reports whether recovery is exact, conditional, or failing, and then explains the blocker in structural terms. It can suggest supported repairs and compare before/after outcomes with explicit status labels.

The studio is intended for branch-supported analysis rather than unrestricted symbolic search. Recommendations are tagged by evidence class (theorem-backed, supported-family theorem, validated benchmark, or conditional guidance), so the user can distinguish proven structure from exploratory heuristics.

Export paths include JSON, CSV series where applicable, markdown report output, and share links. These are designed for reproducibility across branch validation workflows.
