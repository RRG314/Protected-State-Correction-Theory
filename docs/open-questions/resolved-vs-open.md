# Resolved vs Open

## Resolved In This Repository

### Resolved positively
- exact orthogonal protected-subspace recovery
- exact sector recovery for pairwise orthogonal coordinate-compatible sector embeddings
- asymptotic protected-state correction for invariant-split generators
- spectral-gap decay bound in the self-adjoint PSD case
- periodic exact continuous projection via the Leray/Helmholtz anchor
- bounded-domain exact correction on the repository's boundary-compatible finite-mode Hodge family
- restricted-linear minimal augmentation count via the row-space deficiency `δ(O, L; F)`

### Resolved negatively
- overlap / indistinguishability destroys exact recovery
- mixing destroys protected-coordinate preservation in linear flows
- rank-deficient exact correction cannot cover a full disturbance family
- smooth linear flows cannot produce finite-time exact recovery on a nontrivial disturbance space
- overlapping sectors do not support unique exact detection
- universal scalar capacity is not supported by the current program

## Still Open

### High-value open problems
- a robust noisy or admissible-family-enlarged restricted PVRT theorem beyond the current zero-noise exact spine
- boundary-sensitive exact continuous correction beyond the solved finite-mode bounded Hodge subcase
- a mature branch-specific capacity theory beyond lower bounds and the solved restricted-linear deficiency theorem
- a sharper sector-branch formulation beyond the 3-qubit anchor but still tied to real operator language
- a stronger control/observer theorem beyond the invariant-split linear case

### Open but not urgent
- whether one additional exact non-quantum sector system should be added
- whether a limited continuous-time measurement/reset branch belongs in OCP proper

## Closed Or Demoted Directions
- optimizer/ML unification
- broad universal-capacity story
- language implying that GLM is exact
- language implying that generic feedback stabilization is automatically OCP
