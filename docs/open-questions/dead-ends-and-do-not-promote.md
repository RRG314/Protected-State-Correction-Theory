# Dead Ends And Do Not Promote

## 1. Universal Scalar Capacity
Do not revive this without a genuinely new proof architecture. The repository now has enough evidence that forcing one scalar across exact projector, sector, PDE, and control branches will collapse into metaphor.

## 2. Generic Optimizer / ML OCP Claims
Do not promote optimizer language as part of the main theory. At the current evidence level it is, at best, analogy-only.

## 3. “GLM Is Exact” Language
Do not allow any wording that suggests GLM is an exact projector. The entire value of the exact/asymptotic split depends on keeping this distinction sharp.

## 4. Generic Stabilization Equals OCP
Do not treat ordinary damping or feedback stabilization as OCP unless the protected object, disturbance object, and non-corruption condition are all explicit.

## 5. Boundary-Free PDE Extrapolation
Do not speak as if the periodic Helmholtz/Leray result automatically covers bounded domains or complicated boundary data. That is still open.

## 6. Broad Universal PVRT
Do not promote Protected-Variable Recoverability Theory as a universal cross-branch law. The strongest surviving form is the restricted finite-dimensional linear spine. Everything broader must re-earn itself by proof.
