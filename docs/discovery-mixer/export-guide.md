# Discovery Mixer Export Guide

## Available Exports

The mixer integrates with the workbench export system and can produce:

- shareable state encoding
- JSON payload
- CSV export rows
- human-readable Markdown report
- figure / comparison snapshot where the workbench surface supports it

## What The Export Captures

- active lab and mixer mode
- typed family and protected target
- record or architecture choice
- regime verdict
- diagnostics
- recommendations
- before/after comparison data
- evidence level
- reproducibility metadata

## Interpretation Rule

Exports preserve evidence level and scope.
A saved report should make clear whether a result is theorem-backed, family-specific, benchmark-guided, heuristic, or unsupported.
