# Open Problem Program

## Level A — tractable restricted problems

1. Weighted-cost fixed-library strengthening
   - move beyond the solved unit-cost candidate-library theorem
   - prove that genuinely nonuniform sensor costs still do not classify exactness by total budget alone

2. Noisy detectable-only theorem beyond the current restricted-linear witness
   - push the new noisy weaker-versus-stronger separation theorem beyond the current canonical branch example

3. History-versus-observer boundary on richer control families
   - push the exact/asymptotic split beyond the current diagonal toy family

## Level B — serious restricted-to-broad transition problems

1. Structured cost-based augmentation lower bounds
   - move beyond unrestricted linear deficiency to weighted or geometry-constrained augmentations

2. Restricted PDE target hierarchies
   - prove additional weaker-vs-stronger target splits for supported CFD families

3. Branch-specific detectability comparison theorem
   - relate coarsened exact detection, finite-history exact recovery, and asymptotic observer recovery on one restricted class

## Level C — ambitious grounded problems

1. Stability/instability dichotomy beyond exactness
   - develop a theorem program that distinguishes exact impossibility from stable approximate recoverability on broader restricted families

2. Cross-field normal form for target-relative recoverability
   - identify a nontrivial restricted class where observability, decodability, and protected-variable recoverability become formally equivalent in a theorem-grade way

3. PDE-side impossibility without full linearization collapse
   - prove a broader bounded-domain or sparse-measurement impossibility theorem that still remains honest about domain assumptions
