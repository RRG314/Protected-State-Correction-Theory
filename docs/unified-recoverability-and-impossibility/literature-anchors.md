# Literature Anchors

## Purpose

This branch is only useful if it stays honest about where its ingredients come from.
These anchors are not presented as exhaustive literature surveys.
They are the minimum reference map needed to separate:
- standard background,
- repo-local synthesis,
- and plausible contribution candidates.

## Information theory

- Claude E. Shannon, *A Mathematical Theory of Communication* (1948)
  - [IEEE Reach primary-source page](https://reach.ieee.org/primary-sources/a-mathematical-theory-of-communication/)
  - [PDF reprint](https://reach.ieee.org/wp-content/uploads/2023/05/IEEE_REACH_A_Mathematical_Theory_of_Communication.pdf)

Use in this branch:
- channel/output factorization viewpoint,
- message distinction and loss,
- target-relative recovery language.

## Coding theory

- R. W. Hamming, *Error Detecting and Error Correcting Codes* (1950)
  - classical anchor for detect vs correct logic

Use in this branch:
- weaker target vs stronger target,
- detect-but-not-correct structure,
- parity/syndrome ambiguity.

## Inverse problems

- Hadamard well-posedness tradition
  - [Springer overview quoting the standard well-posedness conditions](https://link.springer.com/chapter/10.1007/978-3-030-34413-9_5)
- A. N. Tikhonov, *On the solution of ill-posed problems and the method of regularization* (1963)
  - [MathNet entry](https://www.mathnet.ru/eng/dan28329)

Use in this branch:
- nonuniqueness as fiber collision,
- stability as modulus-of-recoverability behavior,
- approximate inversion versus exact inversion.

## Control / observability

- R. E. Kalman observability/control systems tradition
  - [C.I.M.E. lectures listing](https://link.springer.com/book/10.1007/978-3-642-11063-4)
  - [later observability literature page citing the original 1960/1961 Kalman control-systems work](https://link.springer.com/article/10.1134/S0012266109110061)

Use in this branch:
- finite-history exact recovery,
- asymptotic observer recovery,
- detectability/observability comparison.

## Quantum error correction

- Emanuel Knill and Raymond Laflamme, *A Theory of Quantum Error-Correcting Codes* (1996/1997)
  - [arXiv abstract](https://arxiv.org/abs/quant-ph/9604034)

Use in this branch:
- exact correctability as target preservation plus distinguishable error sectors,
- syndrome logic,
- exact recoverability with a nontrivial error family.

## Repo-local extensions

The repo's own strongest branch-local inputs are:
- restricted PVRT theorem spine,
- same-rank insufficiency theorem,
- minimal augmentation theorem,
- periodic and diagonal threshold laws,
- divergence-only and bounded-domain no-go examples,
- exact-versus-asymptotic branch distinctions.

These are the materials the unified branch is synthesizing and stress-testing.
