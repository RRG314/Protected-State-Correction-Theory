# Naming and Terminology

## Chosen Public Title

**Protected-State Correction Theory**

This is the clearest honest public title because the repository is still theory-first.
It has genuine physics extensions, but physics is not broad enough to dominate the title.

## Chosen Internal Core Term

**Orthogonal Correction Principle (OCP)**

This remains the internal principle name for the exact branch and for the core protected-state correction idea.

Use it as:
- a principle name,
- a theorem-label prefix,
- and a compact internal shorthand.

Do not use it as the only public-facing name.

## Chosen Physics Lane Name

**Physics Extension of Protected-State Correction Theory**

This is the strongest honest framing after the physics audit.

Why this wording won:
- the repo has real physics anchors,
- the physics lane is worth foregrounding as an extension,
- but the repository is not yet broad enough to justify a fully physics-forward master title.

## Chosen Workbench Name

**Protected-State Correction Workbench**

This is clearer than `OCP Workbench` for new readers and still leaves room to mention OCP inside the module and theorem text.

## Rejected Alternatives

### 1. Orthogonal Correction Principle Research Program
Rejected as the primary public title because it makes the acronym carry too much of the explanatory load.

### 2. Protected-State Correction in Physical Systems
Rejected as the master title because the theory remains broader than its current physics lane.

### 3. Orthogonal Correction Principle in Physical Systems
Rejected because the name outruns the current proofs and makes the repo sound more physics-dominant than it really is.

### 4. Universal Correction Theory
Rejected because the repository explicitly does not support universal claims.

## Terminology Rules

### Preferred public wording
- protected-state correction
- exact correction
- asymptotic correction
- protected object
- disturbance family
- correction operator
- correction architecture
- no-go result

### Keep but define
- Orthogonal Correction Principle (OCP)
- sector recovery
- invariant-split generator

### Avoid without qualification
- universal correction
- universal physics law
- exact GLM correction
- generic Hamiltonian fit

## Final Naming Decision

The strongest honest identity is:

> **Protected-State Correction Theory**
>
> Core principle: **Orthogonal Correction Principle (OCP)**
>
> Physics lane: **Physics Extension of Protected-State Correction Theory**
