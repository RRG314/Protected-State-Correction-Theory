# Inverse Problems Through Fiber Language

Fibers:
- causes or states that produce the same observation data

Target:
- full cause, selected latent variable, or protected functional

Exact recovery:
- the target is constant on data fibers

Impossibility:
- nonuniqueness is target variation inside one data fiber

Why this branch matters here:
- it gives a clean target-relative version of nonuniqueness
- full-state recovery may fail even when a weaker target remains fiber-constant

Strongest kept mapping:
- nonuniqueness as target-mixed data fibers

Strongest limit:
- this branch does not claim a universal inverse-problem stability theory beyond the supported modulus/collision-gap lanes.
