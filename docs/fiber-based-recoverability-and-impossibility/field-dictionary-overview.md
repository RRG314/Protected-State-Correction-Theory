# Field Dictionary Overview Through Fiber Language

Every field dictionary in this branch is organized by four questions:

1. what are the fibers?
2. what is the target?
3. what does exact recovery mean?
4. what does impossibility mean?

Each dictionary also keeps one honesty label:
- exact match
- restricted match
- analogy only

## Strongest overall summary

- information theory: exact match at the distinguishability / factorization level
- coding theory: restricted match for detect-vs-correct logic
- inverse problems: exact match for target-relative nonuniqueness
- control: restricted match for finite-history and observer examples
- QEC: restricted exact match on the sector/syndrome anchor
- CFD / PDE: restricted match on coarse-record ambiguity and threshold examples
- MHD: analogy-only except where projector-compatible exact branches are already proved elsewhere in the repo
