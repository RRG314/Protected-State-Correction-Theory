# Coding Theory Through Fiber Language

Fibers:
- error patterns or transmitted states producing the same syndrome / decoding evidence

Target:
- codeword, logical state, or correction label

Exact recovery:
- the target is constant on the syndrome fibers relevant to the chosen decoder

Detectable-only:
- some coarsened property of the error is fiber-constant even when full correction is not

Impossibility:
- one syndrome fiber mixes logically distinct targets or incompatible correction labels

Exact branch relation:
- strongest match is the exact sector / QEC anchor

Strongest kept example:
- syndrome/sector separation in the exact QEC anchor

Strongest limit:
- this branch does not claim every coding-theory detectability notion is identical to the repo's target-coarsening detectability notion.
