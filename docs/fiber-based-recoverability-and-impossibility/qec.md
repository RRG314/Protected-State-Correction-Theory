# QEC Through Fiber Language

Fibers:
- states or errors producing the same syndrome information

Target:
- logical state / code-sector information

Exact recovery:
- the logical target is constant on syndrome fibers under the Knill-Laflamme-compatible sector structure

Impossibility:
- syndrome overlap or sector ambiguity mixes different logical targets in one effective fiber

Best kept mapping:
- exact sector/QEC anchor
- not a claim that all QEC notions collapse to the same branch definition outside that anchor

Strongest kept no-go:
- syndrome overlap or phase-insensitive measurement leaves logical distinctions mixed on one effective fiber
