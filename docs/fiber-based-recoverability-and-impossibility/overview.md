# Fiber-Based Recoverability And Impossibility

This branch studies partial/coarse-record recoverability through:
- record fibers,
- target constancy on those fibers,
- target variation inside those fibers,
- fiber refinement under added measurements,
- and false-positive risk when exact claims are moved outside the family, target, discretization, or architecture that made them true.

Current local branch name:
- `steven/fiber-based-recoverability-and-impossibility`

## What the branch now contains

### Universal backbone
- exact recoverability / identifiability as fiber constancy,
- impossibility as target-mixed fibers,
- detectable-only through target coarsening.

### Strongest negative theorem package
- `OCP-049` rank-only anti-classifier,
- `OCP-050` fixed-library budget-only anti-classifier,
- `OCP-052` family-enlargement false-positive theorem,
- `OCP-053` canonical model-mismatch instability theorem.

### Strongest quantitative hierarchy theorem
- `OCP-051` noisy weaker-versus-stronger separation.

### New seriousness layer from the latest passes
- explicit outside inverse-problem / observability map,
- explicit falsification harness,
- family-restriction, model-mismatch, and discretization false-positive suites,
- stronger workbench warnings about what is and is not identified,
- and a small theorem-linked decision layer for stop / switch / augment / change-architecture guidance.
