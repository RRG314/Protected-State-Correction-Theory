# Fibers And Detectability

## Branch-safe detectability notion

The branch uses a narrow target-relative detectability notion:
- a coarsened target is exact on the current fibers
- the stronger target is not

This is honest and directly connected to the branch theorems.

## What this is not

This is not claimed to be identical to:
- classical control detectability in full generality
- coding-theory detectability in full generality
- one universal cross-field detectability definition

Those broader claims failed and stay demoted.

## Best current use

Use the fiber detectability notion to explain:
- detect-but-not-recover
- weaker-but-not-stronger exactness
- why the workbench can surface meaningful weaker targets without pretending the full target is solved
