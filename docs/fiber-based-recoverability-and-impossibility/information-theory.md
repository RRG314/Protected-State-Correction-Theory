# Information Theory Through Fiber Language

Fibers:
- states that produce the same retained symbol, channel output, or compressed record

Target:
- whatever message, class, or functional the observer wants to reconstruct

Exact recovery:
- target is constant on channel-output fibers

Impossibility:
- two source states share the same output fiber but carry different target values

Exact match to branch language:
- very close at the level of distinguishability and factorization

Strongest kept match:
- factorization through the output map
- ambiguity as target variation on one output fiber

Limit:
- this branch does not claim one universal information-theoretic scalar quantity beyond that exact fiber logic.
