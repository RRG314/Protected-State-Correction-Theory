# Frontier Audit

## What is already strongest

### Universal backbone
- `OCP-030`: exact recoverability is fiber constancy / factorization through the record map.
- `OCP-048`: detectable-only through target coarsening.

These remain the cleanest branch-wide positive statements. They are strong, but they are also closest to standard background in spirit.

### Strongest negative layer
- `OCP-049`: no rank-only exact classifier theorem.
- `OCP-050`: no fixed-library budget-only exact classifier theorem.
- `OCP-051`: noisy weaker-versus-stronger separation theorem.
- `OCP-052`: restricted-linear family-enlargement false-positive theorem.

These are the branch's strongest current theorem-grade novelty candidates because they say something reusable and falsifiable about where broad classifier language fails.

## What is already negative

The branch now has a real anti-overclaim spine:
- same-rank opposite-verdict witnesses,
- same-budget opposite-verdict witnesses,
- exact weak target / impossible strong target splits,
- bounded-domain projector transplant failure,
- periodic hidden-mode threshold failures,
- family-enlargement failure under hidden target-changing directions.

## What is only family-specific

Still restricted by family:
- periodic cutoff / support thresholds,
- diagonal history thresholds,
- qubit phase-loss thresholds,
- bounded-domain compatibility repairs,
- observer versus finite-history exactness.

These are useful and real, but they are not universal identifiability laws.

## What is still weak or under-proved

Not yet strong enough to promote:
- any universal PDE observability equivalence theorem,
- any nonlinear generalization beyond narrow witnesses,
- any universal sensor-placement law,
- any universal discretization-robust exactness criterion,
- any theorem claiming model-mismatch robustness beyond the supported restricted-linear class.

## What is actually worth pushing now

Best current next-step directions:
1. weighted-cost anti-classifier theorems beyond the unit-cost library,
2. restricted nonlinear fiber-compatibility or a clean theorem-failure there,
3. bounded-domain recoverability boundaries under richer families,
4. sensor-geometry theorems that survive beyond coordinate-library witnesses,
5. family-enlargement and model-mismatch certificates on richer admissible classes.
