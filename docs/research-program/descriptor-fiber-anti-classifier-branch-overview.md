# Descriptor-Fiber Anti-Classifier Branch Overview

## Branch Identity

Branch name: **Descriptor-Fiber Anti-Classifier Branch**

Placement: **branch-limited quantitative extraction lane** attached to:
- restricted-linear anti-classifier theorems (`OCP-049`, `OCP-050`),
- family-enlargement/model-mismatch no-go package (`OCP-052`, `OCP-053`),
- fiber-based recoverability/impossibility formalism.

Theory status class: `B` (branch-limited).

## What This Branch Is

This branch turns existing anti-classifier witness sets into a quantitative descriptor-fiber layer. It adds finite-class quantities that are computable from generated witness artifacts:
- descriptor-fiber mixedness (`DFMI`),
- irreducible descriptor-only error lower bound (`IDELB`),
- compatibility lift (`CL`).

Core deliverable type:
- theorem-backed finite witness-class classification,
- validated quantitative extraction,
- no universal promotion beyond supported classes.

## What This Branch Adds

1. A descriptor-fiber purity criterion for perfect descriptor-only classification on finite witness classes.
2. A sharp finite-class lower bound on descriptor-only classification error.
3. A compatibility-lift measurement that quantifies how much structural side-information removes amount-only ambiguity.
4. Canonical quantitative summaries aligned with existing anti-classifier witness files.

## What This Branch Does Not Claim

1. It does not replace OCP core projector or recoverability theorems.
2. It does not claim a universal cross-physics law.
3. It does not promote finite witness evidence to unrestricted continuous theorems.
4. It does not supersede the recoverability or fiber branches; it refines them quantitatively on supported classes.
