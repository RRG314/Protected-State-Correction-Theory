# Master Novelty Pressure Report (OCP)

Date: 2026-04-17

## Scope of this report
This pass applies novelty pressure to OCP-promoted claims and OCP-facing bridge claims by separating:
- standard background,
- standard-but-useful benchmark packaging,
- plausible restricted extensions,
- artifact-risk or overreach classes.

## OCP-promoted claim classes

### Plausible restricted extensions (kept promoted in OCP)
- Rank-only / amount-only exact-classifier failure package (`PROVED ON SUPPORTED FAMILY`).
- Family-enlargement false-positive package (`PROVED ON SUPPORTED FAMILY`).
- Restricted model-mismatch instability envelope (`CONDITIONAL`).

### Standard-but-useful benchmark context (kept as context, not novelty claims)
- Integrable single-state robustness and baseline soliton stability references.
- Modulation-instability regime families and extreme-event background maps.

### Not promotable in OCP now
- Broad dynamic self-organization transfer claims without robust artifact survival.
- Universal OCP/nonlinear unification wording.

## OCP-facing bridge novelty position
- Soliton symmetry non-identifiability is treated as a bounded structural analogue, not a global theorem transfer.
- Same-count opposite-verdict is treated as restricted analogue and remains support-bounded.
- Projection-preservation/no-go parallels remain bridge-level unless operator-class transfer theorems are explicitly proven.
