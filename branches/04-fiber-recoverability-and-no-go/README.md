# Branch 04 — Fiber Recoverability and No-Go

This branch contains the strongest failure structure in the repository. It focuses on fiber-level indistinguishability, same-descriptor opposite-verdict constructions, and explicit anti-classifier results.

The main contribution is not just isolated counterexamples. It is a systematic demonstration that amount-only summaries such as rank or budget fail to classify recoverability, even when those summaries are held fixed across compared systems.

Canonical reading starts at `docs/fiber-based-recoverability-and-impossibility/overview.md`, then `fibers-and-exact-recoverability.md` and `no-go-counterexamples.md`, with descriptor-fiber quantitative extensions documented in `docs/research-program/descriptor-fiber-anti-classifier-branch-overview.md`.

This branch is central to scope control across the entire repo.
