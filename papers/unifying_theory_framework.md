## Abstract
This paper proposes a branch-limited unifying framework across four active research programs: protected-state correction/recoverability (OCP), restricted soliton recoverability, Euler-potential MHD closure/obstruction, and theorem-linked SDS engineering layers. The framework is built by falsification-first synthesis of surviving results rather than by universal projection. The promoted universal core is narrow: exact recoverability as factorization/fiber constancy and impossibility by collision/indistinguishability. Above that core, branch-limited theorems organize structure-over-amount anti-classifier results, family-enlargement fragility, model-mismatch instability, and domain/topology compatibility obstructions. We classify what is unified, what remains branch-specific, and what is analogy-only. At least one abstract theorem survives at universal level, and multiple branch-limited theorem packages survive with explicit scope. The resulting framework explains why these branches belong together mathematically while preserving nonclaims against false unification.

**Keywords:** recoverability, factorization, fiber collision, anti-classifier theorem, model mismatch, family enlargement, domain obstruction, branch-limited unification

## 1. Introduction
Current results across the ecosystem are strong but heterogeneous: exact projector theorems, anti-classifier no-go theorems, bounded-domain obstruction/exactness splits, restricted nonlinear identifiability results, and domain-specific closure theorems in MHD. A useful unifying theory must preserve this strength without flattening differences.

The objective here is not universal physics. It is a mathematically honest unification layer that separates:
1. a universal abstract core,
2. branch-limited strengthening theorems,
3. engineering consequences,
4. analogy-only language.

Main contribution of this paper-form pass:
- a common object language `(X,A,T,M,C,D,~,S)`,
- a tested universal core,
- a classification map for exact/asymptotic/impossible/fragile regimes,
- promoted abstract theorems and explicit failed theorem candidates,
- a nonclaims layer that blocks overreach.

## 2. Master Object Language
We model each branch instance as

`(X, A, T, M, C, D, ~, S)`

where `X` is state space, `A` admissible family, `T` target map, `M` observation map, `C` correction architecture, `D` disturbance/ambiguity structure, `~` nuisance symmetry relation, and `S` side information/augmentation.

This language supports:
- static recoverability questions (`T` from `M`),
- operator preservation/failure questions (`C` on protected classes),
- fragility analysis under family enlargement and model mismatch,
- symmetry-quotient formulations.

## 3. Universal Core
### 3.1 Promoted universal statement
For declared `(A,T,M)`, exact recoverability is equivalent to fiber constancy / factorization:

- `T` exact from `M` on `A` iff `T` is constant on fibers of `M` on `A` iff `T = R ∘ M` on `A`.

### 3.2 Universal no-go corollary
If there exist `x,x' in A` with equal record and distinct target (`M(x)=M(x')`, `T(x) != T(x')`), exact recoverability is impossible.

These two statements are promoted as the universal core because they survive all pressure without branch distortion.

## 4. Classification System
Systems are classified by strongest applicable regime:
1. exact recoverable,
2. exact modulo symmetry,
3. asymptotically correctable,
4. detectable but not exact,
5. impossible by collision/overlap,
6. false-positive under family enlargement,
7. mismatch-unstable,
8. architecture/domain/topology mismatched,
9. branch-external/analogy-only.

Placement is determined by explicit invariants/obstructions, not narrative similarity.

## 5. Shared Invariants and Obstructions
The minimal non-decorative set retained:
- fiber-collision exactness invariant `kappa_0`,

- family-enlargement fragility indicator,

- symmetry-quotient obstruction.

Metrics tied to emergence diagnostics are retained as discovery signals, not promoted as theorem-core invariants.
