## Abstract
We present a branch-limited unifying framework across OCP recoverability/correction theory, restricted soliton recoverability lanes, Euler-potential MHD closure/obstruction lanes, and theorem-linked engineering layers. The universal core is intentionally small: exact recoverability as factorization/fiber constancy on declared admissible families, and impossibility under target-distinguishing collisions. Stronger laws are promoted only as branch-limited theorems: anti-classifier results (no rank-only and no fixed-budget-only exact classifier), family-enlargement fragility, model-mismatch instability, and domain/topology compatibility obstruction. We provide a normalized theorem hierarchy, canonical witnesses/counterexamples, and explicit nonclaims. The result is a defensible framing theory that unifies what is truly shared and keeps non-transferable material separated.

## 2. Definitions and Assumptions
We represent each branch instance by

`(X, A, T, M, C, D, ~, S)`

with state space `X`, admissible family `A`, target map `T`, record map `M`, correction architecture `C`, disturbance/ambiguity structure `D`, nuisance relation `~`, and side-information/augmentation object `S`.

### Exact recoverability
`T` is exactly recoverable from `M` on `A` iff there exists `R` with `R(M(x)) = T(x)` for all `x in A`.

### Fiber collision
A pair `(x,x')` with `M(x)=M(x')` and `T(x)!=T(x')`.

### Exact modulo symmetry
Exactness on quotient classes `A/~` under declared nuisance symmetry.

Assumption blocks are branch-specific and mandatory: restricted-linear (`A={Fz}`), bounded-domain compatibility classes, restricted one-soliton families, and declared MHD ansatz/domain classes.

## 3. Universal Core Theorems
### Theorem U1 (Factorization/Fiber Exactness)
For declared `(A,T,M)`, exact recoverability holds iff `T` is fiber-constant for `M` on `A`, equivalently iff `T` factors through `M` on `A`.

Status: `PROVED`.

### Theorem U2 (Collision No-Go)
If a target-distinguishing fiber collision exists, exact recoverability fails on `A`.

Status: `PROVED`.

### Theorem U3 (Coarsening Monotonicity)
If `q = phi ∘ T` and `T` is exactly recoverable from `M` on `A`, then `q` is exactly recoverable from `M` on `A`; converse need not hold.

Status: `PROVED`.
