# Asymptotic Correction Branch

The asymptotic branch studies correction architectures that suppress disturbance over time rather than remove it in one exact step.

Primary examples:
- `x_dot = -k P_D x`,
- GLM divergence cleaning,
- invariant-splitting control laws.
